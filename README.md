# FinTrack - Expense Tracker App 💰

FinTrack is a modern **personal finance and expense tracking mobile application** built with **Flutter**.  
It helps users manage their daily expenses, track spending, and plan budgets efficiently.

---

## 📱 Features

- 🔐 Secure Authentication using **Supabase**
- 📊 Dashboard showing total balance and expense overview
- ➕ Add and manage expenses
- 🗂 Expense categories
- 📅 Date based expense tracking
- 💰 Budget planning module
- ⚙️ App settings
- 🎨 Clean and modern UI

---

## 🛠 Tech Stack

- **Flutter**
- **Dart**
- **Supabase (Authentication + Database)**
- **GetX (State Management)**
- **Material UI**
